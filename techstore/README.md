# NEXUSTECH — Интернет-магазин компьютеров

## 🚀 Быстрый запуск

Сайт работает как обычные HTML файлы — никаких зависимостей устанавливать не нужно!

### Вариант 1: Открыть напрямую
Просто дважды кликните на файл `index.html` — сайт откроется в браузере.

### Вариант 2: Локальный сервер (рекомендуется)
Для корректной работы всех функций запустите через сервер:

**Через Python:**
```bash
# Python 3
python -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```
Затем откройте: http://localhost:3000

**Через VS Code (расширение Live Server):**
1. Установите расширение "Live Server"
2. Правой кнопкой на index.html → "Open with Live Server"

**Через Node.js:**
```bash
npx serve .
```

## 🛍️ Как добавить/изменить товары

Откройте файл **`js/data.js`**

### Структура товара:
```javascript
{
  id: 23,                    // Уникальный ID (число)
  name: "Название товара",   // Название
  category: "gaming",        // Категория (см. ниже)
  price: 50000,              // Цена в рублях (число)
  oldPrice: 60000,           // Старая цена (null = не показывать)
  image: "https://...",      // URL изображения
  specs: [                   // Характеристики (4 строки)
    "Intel Core i7",
    "RTX 4070 12GB",
    "16GB DDR5",
    "1TB NVMe SSD"
  ],
  badge: "Новинка",          // Бейдж на карточке (null = без бейджа)
  isNew: true,               // Показывать в блоке "Новинки" на главной
  isPopular: false           // Показывать в блоке "Популярные" на главной
}
```

### Доступные категории:
| Значение      | Название на сайте |
|---------------|-------------------|
| `gaming`      | Игровые ПК        |
| `laptop`      | Ноутбуки          |
| `monitor`     | Мониторы          |
| `gpu`         | Видеокарты        |
| `cpu`         | Процессоры        |
| `periphery`   | Периферия         |
| `accessories` | Аксессуары        |

### Как заменить изображения:
Замените `image: "https://placehold.co/..."` на реальную ссылку или путь к файлу:
```javascript
image: "images/products/rtx4080.jpg"  // Локальный файл
// или
image: "https://yoursite.com/img/product.jpg"  // Внешняя ссылка
```

## 📱 Настройка Telegram-бота

Откройте файл **`js/cart.js`** и найдите:

```javascript
const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE';
const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID_HERE';
```

### Как получить токен бота:
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный токен

### Как получить chat_id:
1. Напишите своему боту любое сообщение
2. Откройте: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Найдите в ответе поле `"id"` внутри `"chat"`

### Пример после настройки:
```javascript
const TELEGRAM_BOT_TOKEN = '1234567890:AAFxxx...';
const TELEGRAM_CHAT_ID = '123456789';
```

## 💾 Как работает корзина и заказы

- Корзина сохраняется в `localStorage` браузера
- При оформлении заказ сохраняется в `localStorage` (ключ: `nexustech_orders`)
- Данные заказа выводятся в консоль браузера (F12 → Console)
- Если настроен Telegram-бот — заказ также отправляется в него
